# 🚦 Smart Traffic Congestion Prediction System (ASTRAEA AI)

ASTRAEA AI is an AI-powered traffic monitoring and decision-support system that helps traffic authorities monitor congestion, analyze traffic patterns, and optimize signal timings through intelligent recommendations.

The system collects real-time traffic data such as vehicle count, average speed, queue length, and incidents to calculate congestion levels at multiple intersections. Based on these insights, it suggests optimized traffic signal timings to improve traffic flow and reduce delays. All recommendations are reviewed by traffic operators before implementation.

## Features

- Real-time traffic monitoring dashboard
- AI-based congestion analysis
- Traffic signal optimization recommendations
- Vehicle speed, density, and incident tracking
- Interactive map visualization using OpenStreetMap
- Human-in-the-loop decision support
